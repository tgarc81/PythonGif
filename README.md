# Simple GIF Creator with Python

This project contains a simple script that combines multiple images into an animated GIF file using Python's imageio library. This project is a great starting point for learning the basics of image processing with Python.

## Project Output

When the script is run, it creates a sukuna.gif file using a group images, as shown below:

![Project Demo](https://imgur.com/a/aOP6N3P)

## Setup

Follow these steps to run the project on your local machine.

1. **Clone the repository:**

```sh

**git clone \[[https://github.com/tgarc81/create\_gif.git](https://github.com/tgarc81/PythonGif.git)**

```
2. **Go to the project directory:**

```sh

**cd app**

```
3. **Install the required libraries:**

```sh

**pip install -r imageio**

```

## Usage

After completing the necessary setup, run the following command in the terminal to execute the script:

```sh

python create_gif.py

```

After running this command, a new file named sukuna.gif will be created in your project folder.

## File Structure

- **create\_gif.py**: The main Python script that creates the GIF.

- **sukun-xx.png**: The images used for the GIF

**.gitignore**: Specifies files that should not be tracked by Git (for example, generated GIF files).**
